Weather translation for famers

We have identified and brought to light the problem of adverse effects of weather on agriculture. So weâ€™ve come up with a weather chatbot which uses a weather api to provide live information on the weather and help farmers to make decisions wisely.

How are we doing this?

Our chatbot uses a weather api to fetch the 7-day forecast and will translate the information to the farmers. The farmer will be able to communicate with the chatbot with whatever language he is comfortable with since we will have language translation.
